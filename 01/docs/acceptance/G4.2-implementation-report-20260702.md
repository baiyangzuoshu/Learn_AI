## 3. 实现摘要

### 3.1 Runtime 架构

```
MCP Host
  → tools/call: runtime_status / runtime_scene_tree / ...
    → server.ts (handler dispatch)
      → CDPManager
        → chrome-remote-interface
          → Chrome DevTools Protocol (localhost:9222)
            → Cocos Creator Browser Preview
              → cc.director.getScene() / console.* / Page.captureScreenshot
```

### 3.2 连接管理策略

- **CDPManager** 管理到 Chrome DevTools 的 WebSocket 连接
- **默认端口**：127.0.0.1:9222（可配置）
- **超时**：5000ms（可配置）
- **连接检测**：连接时验证 `cc.director` 存在（确认是 Cocos 预览）
- **断线处理**：ping 失败自动断开，下次调用返回 disconnected
- **控制台缓冲**：通过 CDP `Runtime.consoleAPICalled` 实时捕获，最大 1000 条（可配置）

### 3.3 五个 Runtime Tools

| Tool | 输入 | 未连接输出 | 已连接输出 |
|---|---|---|---|
| `runtime_status` | `{}` | `ok:true, status:"disconnected"` | `ok:true, status:"connected", scene, resolution, fps` |
| `runtime_scene_tree` | `{maxDepth?, maxNodes?}` | `ok:false, RUNTIME_UNAVAILABLE` | `ok:true, tree, totalNodes, truncated` |
| `runtime_node_detail` | `{nodePath}` | `ok:false, RUNTIME_UNAVAILABLE` | `ok:true, name, path, active, position, size, components` |
| `runtime_console_logs` | `{level?, limit?, since?}` | `ok:false, RUNTIME_UNAVAILABLE` | `ok:true, logs[], total, truncated` |
| `runtime_capture_preview` | `{format?, quality?}` | `ok:false, RUNTIME_UNAVAILABLE` | `ok:true, format, base64, sizeBytes` |

### 3.4 安全限制

1. **白名单脚本**：不开放通用 evaluate。只有 3 个预定义表达式模板（scene_tree / node_detail / status）
2. **节点路径校验**：只允许 `[A-Za-z0-9_./\- ]`，最长 512 字符
3. **深度/节点限制**：scene_tree maxDepth 1-8，maxNodes 1-2000
4. **超时**：所有 CDP 操作 5000ms 超时
5. **只读**：不提供修改节点、触发事件、执行任意 JS 的能力
6. **端口绑定**：仅连接 127.0.0.1，不监听 0.0.0.0
7. **错误码**：6 个新错误码，文本可读，不含原始堆栈

## 6. 风险与遗留问题

| 风险 | 影响 | 缓解 |
|---|---|---|
| CDP 连接依赖 Chrome 调试端口配置 | 用户需手动配置 Chrome 启动参数 | 文档说明；Cocos 偏好设置可配置 |
| `cc` 全局对象 API 在 Cocos 版本间可能变化 | 升级 Cocos 版本可能需要调整白名单 | 白名单表达式集中管理，易于修改 |
| 控制台日志缓冲区可能溢出 | 丢失早期日志 | 默认 1000 条，可配置；FIFO 淘汰 |
| `chrome-remote-interface` 第三方依赖 | 供应链风险 | 锁定精确版本 `0.33.2`；无子依赖 |
| GUI 端到端验证未完成 | G4-02/04/05/06 未实际验证 | 已列出人工验证步骤；验证失败可修改 |
